
# Rust Practice

## Rust 실행 환경 구축
* guide : https://www.rust-lang.org/tools/install

### Windows 11
* Rust 공홈에서 ```Rust-up init.exe``` 파일 다운로드 후 설치

### MacOS (monterey)
> curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
>

### IDE
* Jetbrains 사의 **CLion** 제폼을 사용
* Rust 개발을 위해서 Plugin 설치가 필요함(Rust, TOML)
