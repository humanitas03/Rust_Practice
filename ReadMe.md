
# Rust Practice

## Rust 실행 환경 구축
* 선행조건 : 각자 OS에 맞는 C/C++ 개발 환경 구축
* rust install guide : https://www.rust-lang.org/tools/install

### Windows 11
* Microsoft C++ Build Tools가 설치되어 있었음.
* Rust 공홈에서 ```Rust-up init.exe``` 파일 다운로드 후 설치
* guide : https://docs.microsoft.com/ko-kr/windows/dev-environment/rust/setup
